# Hackathon Game

## Setup

### Using Nix

If you are on NixOS, you can use the `shell.nix` provided by running `nix-shell`. If you use direnv, run `direnv allow`.

### Anything else

Run `pip install -r requirements.txt`

## Running the game

### On linux/mac:
`python3 src/main.py`

### On windows

`python src\main.py`

## Controls

Movement: Arrow Keys

Jumping: Space or Up Arrow

Sprinting: Left Shift
